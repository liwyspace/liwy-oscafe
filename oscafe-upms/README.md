## 介绍

演示地址：[http://www.oscafe.org](http://www.oscafe.org) / [http://www.oscafe.net](http://www.oscafe.net)

该用户权限管理系统是基于RBAC模型设计的权限控制通用平台，并提供单点登录、会话管理、和日志管理。

## 系统功能概述

系统组织管理

用户角色管理

资源权限管理

权限分配管理

单点登录（SSO）

用户会话管理
